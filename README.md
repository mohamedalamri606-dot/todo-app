# 📝 To-Do Pro

To-Do Pro is a smart task management web application designed to help users organize their daily tasks efficiently.

The application allows users to add, delete, and complete tasks with a clean and intuitive user interface. It includes advanced features such as automatic task sorting based on priority, smart keyword detection (e.g., "exam", "homework"), real-time search, and a motivational points system.

The app uses local storage to save tasks, ensuring data persistence even after closing the application. It is also converted into a mobile application (APK) using WebView technology, making it accessible on Android devices.

---

## ⚙️ Technologies used:
- HTML  
- CSS  
- JavaScript  

---

## 🚀 Key Features:
- Smart task sorting  
- Priority detection  
- Task filtering and search  
- Undo delete functionality  
- Gamification (points system)  
- Responsive UI design
